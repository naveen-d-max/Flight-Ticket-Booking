# ✈️ SkyBound | Premium Flight Booking
SkyBound is a modern, fully responsive web application designed for a premium flight booking experience. It combines a stunning "Glassmorphic" interface with powerful administrative tools for flight management.
![App Preview](https://raw.githubusercontent.com/YOUR_USERNAME/YOUR_REPO_NAME/main/flight_hero_bg.png)
## ✨ Core Features
- **🔍 Smart Search Engine**: Quickly find flights by origin, destination, date, and travel class.
- **🛠️ Admin Management Panel**: Add, monitor, and remove flights in real-time.
- **🎫 Dynamic Boarding Passes**: Generate professional electronic tickets with passenger details and booking IDs.
- **📊 Data Export**: One-click export of booking records to CSV/Excel for easy management.
- **🎨 Elite Aesthetics**: High-end visual design using HSL color palettes, glassmorphism, and smooth CSS animations.
- **🖨️ Print-Ready**: Built-in print functionality for physical tickets.
## 🛠️ Technology Stack
- **Frontend**: Semantic HTML5 & Modern CSS3 (Vanilla)
- **Icons**: Lucide Icons (CDN)
- **Typography**: Google Fonts (Inter & Montserrat)
- **Persistence**: Browser LocalStorage for persistent data across sessions
- **Animation**: Native CSS Keyframes
